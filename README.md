# Tool Box

로컬에서 `npm run dev`로 실행할 수 있는 탭형 툴박스입니다.

## 기능

- 상단 탭 기반 UI
- QR Maker & Reader
- 기본값 100회로 반복 실행되는 QR 생성 벤치마킹
- 실제 카메라를 열어 라이브 프레임으로 반복 측정하는 리얼 벤치마킹
- 설정 패널의 언어 설정(한국어/영어) 및 테마 설정(다크/라이트/시스템)
- 텍스트 글자 수 / UTF-8 바이트 수 / 줄 수 / 공백 제외 글자 수 확인
- JSON Pretty Viewer

## 실행 방법

```bash
npm run dev
```

기본 포트는 `3000`이며, 브라우저에서 `http://127.0.0.1:3000`으로 접속하면 됩니다.

## 참고

- QR 생성과 리딩은 브라우저에서 CDN으로 로드한 라이브러리를 사용합니다.
- 리얼 벤치마킹은 브라우저의 카메라 권한이 필요합니다.
- 카메라 프레임 인식은 가능하면 `BarcodeDetector`를 사용하고, 지원하지 않는 경우 `jsQR`로 폴백합니다.
